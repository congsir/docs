# 个人博客文档站点

欢迎来到 lccong 的个人博客文档站点！这里收录了我的技术笔记、项目经验和学习心得。

## 📖 关于本站

这是一个基于 VitePress 和 UnoCSS 构建的个人技术博客文档站点，主要用于：

- 记录项目开发经验和技术沉淀
- 分享前端开发相关的知识和技巧
- 展示个人项目成果
- 提供可复用的代码和解决方案

## 🚀 技术栈

- **框架**: Vue 3 + VitePress
- **样式**: UnoCSS
- **部署**: GitHub Pages
- **CI/CD**: GitHub Actions

## 📚 快速导航

使用左侧菜单栏可以浏览本站的所有内容，主要包括以下几个部分：

### 指南

- **简介**: 站点的详细介绍和使用场景
